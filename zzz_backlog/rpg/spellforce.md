# SpellForce Series

RPG/RTS hybrid — hero + party adventuring combined with base-building and army command. Fantasy setting, real-time combat, controllable 3D camera.

## Series Overview

| Entry | Party | RPG vs RTS Balance | Recommendation |
|-------|-------|--------------------|----------------|
| **SpellForce 1** (Platinum) | Hero + small party | Full RTS base/army + party RPG | Good hybrid starting point, dated graphics |
| **SpellForce 2** (Shadow Wars + DLC) | Hero + companions | More polished, balanced RTS/RPG | **Best for DS fans** — often grouped with DS2 |
| **SpellForce 3** (+ Soul Harvest, Fallen God) | Hero + up to 3 companions | Modern visuals, RTS can feel basic in SP | Strong if you want newer graphics |
| SpellForce 3: Versus | Limited | Pure RTS skirmish, no campaign | Free — low-risk way to test combat |

## How it compares

**Pros (vs. Dungeon Siege):**
- Party segments feel similar: real-time combat, loot, exploration, fantasy questing
- Strong hero progression with skill trees and gear for each character
- Camera is flexible (rotate/zoom/pan), not locked

**Cons (vs. Dungeon Siege):**
- RTS layer adds scale but dilutes seamless party dungeon crawling
- No tactical pause like IWD/PoE — closer to DS's action flow
- Army management can dominate some missions

**Bottom line:** If the RTS hybrid sounds fun, start with **SpellForce 2**. If you want pure party RPG without base-building, look at IWD or PoE instead.
