# NWN: Enhanced Edition — Fit Analysis

**Verdict: Weak fit.** Great solo-hero D&D RPG, but lacks the party control and top-down view core to the Dungeon Siege style.

## What it gets right

| Criterion | Status |
|-----------|--------|
| Fantasy setting | High-fantasy D&D |
| RPG depth | Excellent (classes, feats, skills, prestige classes, loot) |
| Combat | Real-time with pause |
| Platform | PC — Steam/GOG, EE runs great on modern systems |
| Mod support | Massive community content library |

## Where it falls short

| Criterion | Issue |
|-----------|-------|
| **Party control** | Solo hero + 1-2 AI henchmen with basic commands. No direct multi-character management like DS/IWD/BG. EE patches add some DM-style commands but it's still fundamentally single-character focused. |
| **Camera/view** | 3D chase cam, not isometric. Can zoom out or mod for pseudo-top-down, but not native. NWN**2** has proper strategy/top-down modes. |

## Better alternatives

- **Icewind Dale EE** — full 6-person custom party, true isometric, heavy dungeon crawl
- **Pillars of Eternity** — isometric, full party management, deep tactical RTwP
- **Drakensang** — strong party control, flexible camera, fantasy RTwP
